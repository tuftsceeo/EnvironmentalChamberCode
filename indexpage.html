Environmental Chamber Index Page

By Walker Wind 6/13/19

This website is being used as part of the Brass Acid Bleed Project.
There are a couple pages that display information about the conditions of
the environmental chamber, which contains samples of brass subject to
specified heat and humidity conditions. Further improvements to this site
include pushing it from a local web serve to a globally accessed website,
allowing the data to be updated live and displayed live, and controlling
the conditions of the chamber from user input. The following pages are
extensions to the current website.



/textcharta : Displays a text chart of humidity data a from the latest test


/textchartb : Displays a text chart of humidity data b from the latest test


/chartdata : Displays the data on a graph from the latest test
